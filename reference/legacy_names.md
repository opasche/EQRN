# Internal renaming function for back-compatibility

Internal renaming function for back-compatibility

## Usage

``` r
legacy_names(eqrn_fit, classes = NULL)
```

## Arguments

- eqrn_fit:

  EQRN fitted object.

- classes:

  If provided, overrides classes of `eqrn_fit`.

## Value

The `eqrn_fit` object with updated attribute names and classes.
