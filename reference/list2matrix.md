# Convert a list to a matrix

Convert a list to a matrix

## Usage

``` r
list2matrix(lst, dim = c("row", "col"))
```

## Arguments

- lst:

  A list.

- dim:

  One of `"row"` (default) or `"col"`.

## Value

The list converted to a matrix, by stacking the elements of `lst` in the
rows or columns of a matrix.
