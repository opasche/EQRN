# Default batch size (internal)

Default batch size (internal)

## Usage

``` r
batch_size_default(tensor_dat, batch_size = 256)
```

## Arguments

- tensor_dat:

  A
  [`torch::dataset()`](https://torch.mlverse.org/docs/reference/dataset.html).

- batch_size:

  An initial batch size, by default `256`.

## Value

The fixed batch_size.
